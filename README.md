# Offline Sync Notes App

## Features
- Create notes offline
- Sync with Firebase when online
- Queue-based sync system

## Tech Stack
- Flutter
- Hive (Local DB)
- Firebase Firestore

## How to Run
1. Clone repo
2. Run `flutter pub get`
3. Run app
